# About
<p>Simple mvc todo-app build with express, mysql, sequelize and handlebars

## Install

```bash
$ git clone https://github.com/RubenFontes/simple-financial-system 
```

```bash
$ cd simple-financial-system
```

```bash
$ npm install
```

## Run

```
npm start
```
